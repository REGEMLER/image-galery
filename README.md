# image-galery

## Описание проекта
Unsplash один из крупнейших поставщиков фотографий в Интернете. Unsplash API позволяет получать бесплатные фото, тематику которых можно указать.

## Задача
Cоздать приложение отображающее полученные от API фото. Добавить приложению поиск. При вводе поискового запроса изменяють фото, которые отображаются в приложении
